Summary: Latitude Vs. X

Sample The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 random cities.

After assembling the dataset, we used Matplotlib to plot various aspects of latitude versus weather. Factors included: max temperature, humidity, cloudiness, and wind speed. This site provides the source data and visualizations created as part of the analysis, as well as any trends and correlations witnessed in the course of the analysis.
